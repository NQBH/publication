**Code & Data Availability.** To facilitate full reproducibility and practical application of our theoretical framework, all polynomial-time algorithms and numerical simulations have been fully implemented in C++ and Python. The complete open-source codebase, computational verifications of the fractional bounds, and supplementary datasets are publicly accessible via the authors' GitHub repository at \url{https://github.com/NQBH/publication/nonideal_covering_rough_set}.
